through the Chainlit UI (`:8501`), captured end-to-end so the `/ask/stream`
stage events are visible — the per-stage progress list updating
("Analizando la consulta" → "Recuperando documentos del DOGV" → … →
"Redactando la respuesta") and the final answer with its citations.

Keep it short (~30–45 s real time, sped up 2–4× if needed), light (<10 MB,
GitHub renders README images up to that comfortably), and free of any
personal data in the prompt.
