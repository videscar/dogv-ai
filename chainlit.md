# Assistente DOGV

Fes consultes sobre el DOGV i revisa sempre les cites aportades per la resposta.

## Recomanacions d'ús

- Formula la pregunta amb el màxim de context possible (tema, organisme, data aproximada).
- Usa una pregunta per torn per millorar precisió i cites.
- Verifica sempre les fonts enllaçades abans de prendre decisions.
